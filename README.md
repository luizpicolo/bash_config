# Mac Configuration

My mac config

# how to use

    git clone git@github.com:luizpicolo/mac_config.git ~/mac_config
    cd ~/mac_config
    cp .bash_aliases ~/

Edit `.bash_profile` and add

    # Bash to Alias
    source ~/.bash_aliases

Execute

    source ~/.bash_profile
